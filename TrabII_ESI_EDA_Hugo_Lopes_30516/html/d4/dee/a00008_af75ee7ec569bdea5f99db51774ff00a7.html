Node2 * readFile ( const char * ficheiro,
int * totV,
int * totA,
bool * res )

Lê uma matriz de um ficheiro onde as linhas representam vértices e as colunas representam adjacências, e cria uma lista ligada de nós representando os elementos da matriz.

Parâmetros
ficheiroO nome do ficheiro a ser lido.
totVApontador para a variável que armazenará o número total de vértices (linhas).
totAApontador para a variável que armazenará o número total de adjacências (colunas).
resApontador para uma variável bool onde será armazenado o resultado da operação. Será definido como verdadeiro se a leitura for bem-sucedida, ou falso caso contrário.
Retorna
Apontador para o início da lista ligada de nós, ou NULL se a leitura falhar.
