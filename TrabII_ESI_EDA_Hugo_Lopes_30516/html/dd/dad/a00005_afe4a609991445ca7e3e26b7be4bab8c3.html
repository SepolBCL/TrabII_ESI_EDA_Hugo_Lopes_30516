int CountPaths ( Graph * g,
int src,

int pathCount )

Conta o número de caminhos possíveis entre dois vértices num grafo.

Esta função calcula o número de caminhos possíveis entre dois vértices num grafo não direcionado.

Parâmetros
gO apontador para o grafo onde será feita a contagem de caminhos.
srcO ID do vértice de origem.
dstO ID do vértice de destino.
pathCountO número atual de caminhos contados.
Retorna
O número total de caminhos possíveis entre os vértices de origem e destino.
