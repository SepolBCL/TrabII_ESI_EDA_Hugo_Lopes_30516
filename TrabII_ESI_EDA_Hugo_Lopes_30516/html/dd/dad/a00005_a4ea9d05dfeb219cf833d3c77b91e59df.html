void ShowAllPath ( Best b,
int n,
int v )

Mostra todos os caminhos mais longos a partir de um vértice inicial.

Este programa imprime todos os caminhos mais longos a partir de um vértice inicial, usando as informações fornecidas pela estrutura Best, que contém os caminhos mais longos e os vértices predecessores para cada vértice no grafo.

Parâmetros
bA estrutura Best contendo os caminhos mais longos e os vértices predecessores.
nO número total de vértices no grafo.
vO vértice inicial a partir do qual os caminhos serão mostrados.
Retorna
Void (sem retorno).
