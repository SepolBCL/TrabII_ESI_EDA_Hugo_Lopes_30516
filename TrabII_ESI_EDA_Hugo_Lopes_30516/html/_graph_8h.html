Algoritmo de Dijkstra.

Este algoritmo calcula o caminho mais curto de um vértice inicial para todos os outros vértices em um grafo ponderado não direcionado.

Parâmetros
gO apontador para o grafo.
nO número total de vértices no grafo.
vO vértice inicial para o qual será calculado o caminho mais curto.
Retorna
Retorna uma estrutura Best contendo o caminho mais curto e os vértices predecessores para cada vértice no grafo.
